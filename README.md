# Moremi Digital Skills

A public library of Claude skills for founders and senior leaders. Each skill is a structured instruction set that loads into Claude to improve performance on a specific, high-value task.

These skills are built to a single standard: every skill in this library should be something you would pay for if someone else had built it first.

---

## What is a Claude skill?

A skill is a file you upload to Claude that gives it a precise set of instructions for a specific task. Rather than prompting Claude from scratch each time, a skill loads a tested framework directly into the conversation, so Claude applies it consistently and completely without you having to specify how.

Skills work in Claude.ai on any plan. You do not need technical knowledge to use them.

---

## How to use a skill

1. Open [Claude.ai](https://claude.ai)
2. Start a new conversation
3. Click the paperclip or attachment icon
4. Upload the `SKILL.md` file from the skill folder you want to use
5. Tell Claude what you need. The skill will guide its approach automatically.

You can upload more than one skill file into the same conversation if your task spans multiple areas.

## Skills in this library

### [Authentic Writing Voice](./authentic-writing-voice/)
A prohibitions-based skill that removes AI writing patterns from any content before it is published. Covers structural patterns, sentence-level constructions, punctuation, and register. Applies to LinkedIn posts, articles, proposals, email copy, and any prose where voice and authenticity matter. Use it for generating new content or auditing existing drafts.

### [Decision Audit](./decision-audit/)
A structured audit framework for examining the quality of reasoning behind a past decision, independent of how it turned out. Works through decision capture, assumption mapping, information audit, reasoning quality, bias assessment, risk assessment, outcome separation, pattern identification, reconstruction, and framing for others. Domain-agnostic. Applies to strategic, operational, financial, hiring, and product decisions.
